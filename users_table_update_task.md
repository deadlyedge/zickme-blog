# Users表结构优化任务

## 需求
修改users表，允许注册用户修改自己的资料，允许管理员修改邮箱，添加用户最近登录时间，添加用户comments数量。

## 修改内容

### 1. 新增字段
- [ ] **lastLoginAt** - 用户最近登录时间
- [ ] **commentsCount** - 用户评论数量
- [ ] **profile** - 用户资料信息（昵称、头像、个人简介等）

### 2. 权限设置
- [ ] **用户可修改字段**
  - 用户名 (username)
  - 密码 (通过认证系统)
  - 资料信息 (profile)
  
- [ ] **管理员可修改字段**
  - 邮箱 (email)
  - 用户角色 (roles)
  - 用户资料信息 (profile)

### 3. 字段定义
- [ ] **lastLoginAt**: timestamp 类型，自动记录登录时间
- [ ] **commentsCount**: number 类型，显示用户评论总数
- [ ] **profile**: group 类型，包含昵称、头像、个人简介等子字段

## 实现步骤
1. 查看现有的Users表结构
2. 添加新字段定义
3. 设置适当的访问权限
4. 创建数据库迁移
5. 更新相关类型定义
